  check-in flow, iCal sync (Airbnb/Booking.com), seasonal pricing engine, Italian police reporting (Alloggiati SOAP
  integration). Live pilot May 2026.

  **Stack:** Next.js · TypeScript · PostgreSQL · Supabase · Node.js · CI/CD

  ---

  ### Agentic Fintech Ecosystem — Autonomous Trading System
  Multi-agent trading system. Rust sensory array ingests and normalises market data (Polygon.io, NBBO) into QuestDB via
  raw ILP. Python GaussianHMM regime detector publishes market state via Redis pub/sub. LangGraph Blue/Red/Judge debate
  graph synthesises trade signals with Mistral + Claude.

  **Stack:** Rust · Python · LangGraph · QuestDB · Redis · Docker · Anthropic Claude API

  [View repo →](https://github.com/alijendoubi/agentic-fintech-ecosystem)
